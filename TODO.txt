You are an expert software architect and full-stack developer specializing in creative tools for performing arts. Build a complete, functional web-based dance formation planning application that uses AI to help choreographers design formations and transitions for groups of dancers.

## Architecture & Codebase Structure

Design the codebase with a **strict frontend/backend separation** to support parallel development by two agents with minimal merge conflicts:

- **Frontend:** React application in its own directory (`/client`), using an interactive SVG or canvas stage grid with drag-and-drop dancer repositioning. All UI state, components, and rendering logic live here. Communicate with the backend exclusively via a well-defined REST API (or WebSocket where needed for animation).
- **Backend:** Node.js or Python (whichever best suits the AI integration) in its own directory (`/server`), exposing clearly documented API endpoints. All AI formation logic, center time calculations, and data persistence live here.
- Define a clear **API contract** (endpoint list with request/response shapes) as part of the setup documentation so both agents can work independently without stepping on each other.
- Avoid shared configuration files or cross-directory imports that would cause merge conflicts.

## AI-Assisted Formation Generation

The AI layer (backend) should:

- Generate multiple formation snapshots across the song's timeline, triggered by musical phrases or user-defined timestamps
- Suggest smooth transition paths between formations (minimizing crossing paths, balancing stage weight)
- Generate named formation effects (e.g., V-shape, arc, cluster, split, diagonal) with labeled snapshots

## Center Time Tracking & Balancing

- **Track center time** per dancer — defined as time spent at or near the red X focal point across all formation snapshots
- When a choreographer clicks any dancer icon, display a panel showing that dancer's **total center time and percentage share** relative to the full group
- Provide a **center time balancing tool** to adjust formations so center time is distributed according to the choreographer's intent — either equitably or weighted toward specific individuals
- **Flag dancers** who are significantly over- or under-represented in center time relative to the choreographer's target distribution
